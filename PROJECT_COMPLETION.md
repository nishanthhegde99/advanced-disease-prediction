## 📊 UNIFIED SYSTEM DETAILS

### **Confidence Calculation**
```
Formula: (ML × 0.70) + (Rules × 0.15) + (Temporal × 0.15)

Example:
ML Confidence:      85.0%
Rule Confidence:    78.0%
Temporal Score:     82.0%

Calculation:
(85.0 × 0.70) + (78.0 × 0.15) + (82.0 × 0.15) = 83.5%
+ Symptom Bonus (+8%) = 91.5% ✅ VERY_HIGH
```

### **Clinical Trust Levels**
```
90%+  →  VERY_HIGH       ✅  PROCEED_WITH_CONFIDENCE
80%+  →  HIGH            ✅  PROCEED_WITH_VERIFICATION
70%+  →  MODERATE        ⚠️   CONSIDER_ALTERNATIVES
65%+  →  FAIR            ⚠️   REQUIRE_INVESTIGATION
<65%  →  LOW             ❌  INSUFFICIENT_DATA
```
